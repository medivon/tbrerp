# Order Completion Is Shipment Completion

Order Completion means all required shipments for the Order have been closed, not that every payment or COD audit has been financially reconciled. Financial Follow-up stays separate so delivery work can close cleanly while payment audit, COD differences, refunds, retained deposits, or credits remain visible in their own management view.

## Considered Options

- Block Order completion until financial confirmation is fully reconciled.
- Close the Order when shipments are complete and track financial issues separately.

## Consequences

Admins can finish operational delivery without waiting for accounting audit, but reports must show financial follow-up separately so unpaid or mismatched amounts are not hidden.
